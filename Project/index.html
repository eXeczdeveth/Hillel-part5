Разновидности селекторов

Селекторы в CSS ( Cascading Style Sheets)
используются для выбора элементов на веб-странице
и задания для них стилей. Ниже приведены примеры
различных типов селекторов и описание того
какие элементы будут выбраны на странице.

    Селектор по названию элемента: div,h1,p и.т.д
      Пример селектора: P
    • Описание:Этот селектор выбирает все элементы P на странице, то есть абзацы.
    Селектор по индификатору (id):
      Пример селектора: #header
    • Описание:Этот селектор выбирает элемент с уникальным идентификатором "header"
      например, div id="header"
    Селектор по классу: .class
      Пример селектора: .button
    • Описание:Этот селектор выбирает все элементы с классом"button"
      Например, button class="button"на странице,то есть абзацы.
    Селектор по нескольким классам:
      Пример селектора: .red.text
    • Описание:Этот селектор выбирает все элементы,
      которые имеют одновременно классы "red" и "text"
      Например, span class="red text".
    Селектор по вложенности:
      Пример селектора: ul li
    • Описание:Этот селектор выбирает все элементы li
      которые находятся внутри элемента ul
      Это используется для стилизации списков.
